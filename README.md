Welcome to the Cat Cafe!
---
The *"Cat Cafe"* website - is an interactive web project that consists of a fictional cat-themed cafe. This project demonstrates practical web development skills using HTML, CSS, BootStrap, JS and jQuery. 
The site was created to provide a user-friendly, responsive interface and also includes several interactive components such as animation, user interaction through dynamic tables, real-time search, and a variety of visual additions and effects. All of this is designed for a comfortable and improved use of the site for users.
---
**Pages and Features:**

**1)Home Page.**

-Introduction to the Cat Cafe for visitors.

-Includes an interactive image carousel.

-Features "Learn More" modal information.

-Boxes, which includes essential website sections.

 **2)Menu Page.**

-Organized menu with food, drinks and cat snack columns.

 **3)Our Cats Page.**

-Cat Gallery, displays a list of cats with their images and description.

-Includes a live search function, for clear search of exact cat.

**4)Event Page.**

-Showcases upcoming cafe events.

-Hover animation and dynamic content loadng.

**5)Reservation Page.**

-Validated booking form.

-CRUD-enabled table.

-Schedules with visiting time and number of available places.

**Technologies used:**

1)HTML5 - content structure.

2)CSS and Bootstrap - layout and responsive design.

3)JS and jQuery - interactivity.

**Learning Goals:**

1)DOM Manipilation practice by using JS and jQuery.

2)Form Validation and CRUD operations.

3)Use responsive web design principles.

4)Create smooth animations and effects for better UX.
